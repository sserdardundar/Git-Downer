GitHub Repository Downloader

This is a Chrome extension that allows you to download GitHub repositories or specific subdirectories as ZIP files directly from the GitHub interface.

To use this extension:

  1. Install it in Chrome
  2. Navigate to any GitHub repository
  3. Click the "Download Repository" button added to the GitHub interface
  4. Or use the extension popup by clicking its icon in the toolbar

The extension will download the repository or specific directory as a ZIP file.
